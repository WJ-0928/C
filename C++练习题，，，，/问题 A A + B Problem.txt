题目描述

Calculate A + B.


输入

Each line will contain two integers A and B. Process to end of file.


输出

For each case, output A + B in one line.


样例输入
1 1
2 2
3 3


样例输出
2
4
6






#include<iostream>
using namespace std;
int main()
{
	int A,B;
	while(cin>>A>>B)
	cout<<A+B<<endl;
	return 0;
 }
